**Ready to build smarter apps and AI agents with GitHub Copilot?**

GitHub Copilot Dev Days is coming to **Trivandrum on 25 April 2026**, and you’re invited.

Join **Trivandrum Python Community**, in partnership with **GitHub**, for a high-energy, hands-on community event where
developers and tech enthusiasts come together to explore the power of **GitHub Copilot** in the real world.

This is part of **GitHub Copilot Dev Days**, a global in-person user group initiative running from **March 15 to May 15,
2026**. The Trivandrum stop is all about practical learning, live demos, building sessions, and discovering how Copilot
can help you create faster, better, and smarter.

Come to learn how to build **apps and AI agents using GitHub Copilot**, meet fellow builders, take part in the coding
action, and have a great time doing it.

**Pizza is on us. GitHub swag is up for grabs.**

See you there.
